What this is

This tool computes travel‑time isochrones (areas reachable within X minutes) using OpenRouteService (ORS) on OpenStreetMap data. Choose a profile, set travel times, click the map to set an origin (or paste coordinates), then press Generate.

Where to add your ORS key: in the script below replace YOUR_ORS_API_KEY_HERE with your key.
